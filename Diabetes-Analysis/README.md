# 🩺 Diabetes Analysis

## 🔍 Overview

This project analyzes a diabetes dataset and builds a Logistic Regression model to predict whether a patient has diabetes.

## 📊 Steps Performed

* Exploratory Data Analysis (EDA)
* Outlier Detection using Boxplots
* Data Cleaning
* Feature Selection
* Logistic Regression Model

## 📈 Results

* Accuracy: **78%**
* AUC Score: **0.82**

## 📁 Files in this folder

* Dataset (Excel/CSV)
* Analysis code (Python)
* Final report (PDF)
* Visualizations

## 📌 Key Insights

* Plasma glucose, BMI, and Age are strong predictors
* BP and Insulin are less significant
