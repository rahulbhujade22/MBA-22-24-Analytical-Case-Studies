# 📊 Regression Evaluation Metrics Analysis (Axis Project)

## 🔍 Overview

This project focuses on evaluating the performance of regression models using key statistical metrics in Excel.

## 🎯 Objective

To understand and calculate different regression evaluation metrics for assessing model accuracy and performance.

## 📊 Dataset

* Structured dataset used for regression analysis
* Includes actual vs predicted values

## 🛠 Tools Used

* Microsoft Excel

## ⚙️ Methodology

### 📌 Steps Performed:

* Applied regression model outputs
* Compared actual vs predicted values
* Calculated error terms

### 📐 Evaluation Metrics Used:

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)
* R-squared (R²)

## 📈 Key Insights

* RMSE helps in understanding prediction error magnitude
* R² indicates how well the model explains variance
* Lower error values indicate better model performance

## 📁 Files Included

* Excel file with regression calculations
* Evaluation metrics computation

## 📌 Conclusion

The project demonstrates how regression models are evaluated using statistical metrics, helping in selecting the best-performing model.
