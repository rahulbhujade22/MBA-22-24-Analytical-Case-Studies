# 🛒 Association Rule Mining (Market Basket Analysis)

## 🔍 Overview

This project applies association rule mining techniques to identify relationships between items in transactional data using Market Basket Analysis.

## 🎯 Objective

To discover patterns and associations between products that are frequently purchased together, helping businesses improve cross-selling and recommendation strategies.

## 📊 Dataset

* Transactional dataset (Excel)
* Contains item purchase records

## 🛠 Tools Used

* Microsoft Excel
* Data Mining Techniques

## ⚙️ Methodology

### 📌 Steps Performed:

* Data preparation and cleaning
* Conversion of transactional data into suitable format
* Identification of itemsets

### 📐 Techniques Used:

* Association Rule Mining
* Market Basket Analysis

### 📊 Metrics Calculated:

* Support → Frequency of itemset occurrence
* Confidence → Likelihood of items being purchased together
* Lift → Strength of association between items

## 📈 Key Insights

* Identified frequently purchased item combinations
* Discovered strong association rules between products
* Useful for:

  * Product placement
  * Recommendation systems
  * Cross-selling strategies

## 📁 Files Included

* Excel file with association calculations
* Derived rules and insights

## 📌 Business Applications

* Retail & e-commerce recommendations
* Inventory management
* Sales optimization
* Customer behavior analysis

## 📌 Conclusion

Association rule mining helps businesses understand customer purchasing behavior and optimize product strategies for better sales performance.
