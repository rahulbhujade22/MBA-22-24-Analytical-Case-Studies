# 🚚 Logistics Stock Price Forecasting using ARIMA

## 🔍 Overview

This project applies time series forecasting techniques (ARIMA) to predict stock prices of logistics companies such as DHL, FedEx, and JIT Infra.

## 🎯 Objective

To analyze historical stock price data and build forecasting models for predicting future trends.

## 📊 Dataset

* Historical stock price data for:

  * DHL
  * FedEx
  * JIT Infra
* Time-based data (monthly/weekly)

## 🛠 Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Statsmodels (ARIMA)

## ⚙️ Methodology

### 📌 Steps Performed:

* Time series visualization
* Data cleaning and formatting
* Conversion of date column into datetime format
* Setting time index for analysis 

### 📉 Modeling:

* Stationarity check
* ACF & PACF analysis
* ARIMA model building
* Forecast generation

## 🤖 Models Used

* ARIMA (Autoregressive Integrated Moving Average)

## 📈 Key Insights

* Time series patterns identified for each company
* Seasonal trends observed in logistics stock data
* Forecasting helps in understanding future price movement

## 📊 Files Included

* Python scripts:

  * DHL ARIMA 
  * FedEx ARIMA 
  * JIT Infra ARIMA 
* Dataset files
* Analysis outputs

## 📌 Conclusion

ARIMA models effectively capture time-based patterns and can be used for forecasting stock prices in logistics companies.
